# Restored Fountain Tabernacle

A modern, premium church landing page for Restored Fountain Tabernacle ministry.

## Structure

- `index.html` - Semantic HTML with sections ready for React component extraction
- `css/styles.css` - Design tokens, responsive layout, animations
- `js/main.js` - Hero slideshow, scroll animations, smooth scroll
- `assets/logo.png` - Church logo

## Running Locally

```bash
npm install
npm run dev
```

## Design System

- **Primary:** Deep Royal Blue (#0b2d5c)
- **Accent:** Metallic Gold (#c9a227)
- **Typography:** Cormorant Garamond (headings), Source Sans 3 (body)

## Future Integration

Sections use semantic elements (`<section>`, `<article>`, `<nav>`) and clear class naming for easy migration to React components and Firebase-backed content.
